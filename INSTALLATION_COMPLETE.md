## 🚀 الخطوات التالية

### الخطوة 1: إضافة منتجات

1. اذهب إلى **لوحة الإدارة** → **المنتجات**
2. أضف منتج جديد واختر فئة **"ألعاب وتطبيقات"**
3. مثال:
    - الاسم: "Fortnite $50"
    - السعر: 300 (دولار )
    - الفئة: ألعاب وتطبيقات
    - الصورة: صورة اللعبة

### الخطوة 2: إعداد Telegram (مهم!)

بدون هذه الخطوة لن تصل الإشعارات:

1. افتح Telegram وابحث عن `@BotFather`
2. أرسل `/newbot` واتبع الخطوات
3. احصل على **Token** (مثال: `123456789:ABCDEFGHijkl_MNOpqrstu`)
4. حصول على **Chat ID**:
    - ابحث عن `@userinfobot`
    - أرسل رسالة
    - انسخ الرقم من الرد
5. افتح ملف `.env` وأضف:

```
TELEGRAM_BOT_TOKEN=123456789:ABCDEFGHijkl_MNOpqrstu
TELEGRAM_CHAT_ID=123456789
```

6. شغّل:

```bash
php artisan config:clear
php artisan cache:clear
```

### الخطوة 3: اختبار النظام

1. اذهب إلى `http://yoursite.com/games-and-apps`
2. اضغط على منتج وثم "🎮 شحن الرصيد"
3. أملأ النموذج واضغط "إرسال الطلب"
4. ستظهر لك رسالة نجاح
5. تحقق من Telegram ستجد الإشعار! 📱

## 🎯 كيفية الاستخدام

### للعملاء (الزوار):

1. الذهاب للصفحة الرئيسية
2. اختيار **ألعاب وتطبيقات** من الأزرار العلوية
3. اختيار لعبتهم
4. الضغط على **🎮 شحن الرصيد**
5. ملء النموذج بالمعلومات المطلوبة (ID, صورة الإثبات, رقم العملية)
6. الإرسال

### للمسؤول (أنت):

1. الذهاب إلى **لوحة الإدارة** → **🎮 طلبات شحن الألعاب**
2. مراجعة الطلبات الجديدة
3. فتح كل طلب والتحقق من:
    - معرف اللاعب
    - صورة إثبات التحويل
    - رقم العملية
    - بيانات التواصل
4. تحديث الحالة:
    - **قيد الانتظار** → بدء المعالجة
    - **قيد المعالجة** → شحن الرصيد
    - **مكتمل** → الانتهاء
    - **مرفوض** (إذا كانت البيانات غير صحيحة)
