# شرح نظام الإلكترونيات والأجهزة مع طلب WhatsApp

## 📋 المميزات الجديدة:

### 1. قسم الإلكترونيات والأجهزة الجديد

- قسم مخصص للمنتجات الإلكترونية
- تصميم برتقالي أنيق مختلف عن التصاميم الأخرى
- صفحة عرض منفصلة لكل منتج

### 2. نظام الطلب عبر WhatsApp

- كل منتج إلكترونياتي يمكن طلبه عبر WhatsApp
- النموذج يتطلب:
    - اسم العميل
    - رقم الهاتف (مع رمز الدولة مثل +966)
    - الكمية المطلوبة
- حساب تلقائي للمبلغ الإجمالي
- رابط WhatsApp مباشر للتواصل

### 3. حفظ الطلبات في قاعدة البيانات

- جميع طلبات WhatsApp يتم حفظها بقاعدة البيانات
- تتبع حالة الطلب (قيد الانتظار، مرسل، مؤكد، مكتمل، ملغى)
- لوحة تحكم إدارية لمراقبة الطلبات

## 🛠️ طريقة الاستخدام:

### أولاً: تثبيت قاعدة البيانات

```bash
php artisan migrate
```

### ثانياً: إضافة البيانات الأولية (اختياري)

```bash
php artisan db:seed --class=ElectronicsCategorySeeder
php artisan db:seed --class=ElectronicsProductsSeeder
```

### ثالثاً: إضافة منتجات النسخة الإدارية

1. اذهب إلى `/admin/products/create`
2. أضف منتج جديد مع ملء الحقول:
    - **الاسم**: اسم المنتج
    - **الوصف**: وصف المنتج
    - **القسم**: اختر "الكترونيات واجهزه"
    - **السعر**: سعر المنتج
    - **الصورة**: صورة المنتج (اختياري)
    - **يتطلب WhatsApp**: ضع علامة صح ✓
    - **رقم WhatsApp**: أدخل رقم الاتصال (مثال: 966501234567 بدون +)

## 📱 رقم WhatsApp الافتراضي:

```
966501234567
```

يمكنك تغييره لكل منتج على حدة

## 🌐 الروابط الجديدة:

| الرابط                   | الوصف                      |
| ------------------------ | -------------------------- |
| `/electronics`           | صفحة الإلكترونيات الرئيسية |
| `/electronics/{id}`      | صفحة منتج إلكتروني         |
| `/admin/whatsapp-orders` | إدارة طلبات WhatsApp       |
| `/api/whatsapp-orders`   | API لإنشاء طلب (POST)      |

## 📊 جداول قاعدة البيانات الجديدة:

### 1. `whatsapp_orders` - جدول طلبات WhatsApp

```
- id: معرف الطلب
- product_id: معرف المنتج
- customer_name: اسم العميل
- customer_phone: رقم هاتف العميل
- quantity: الكمية
- total_price: المبلغ الإجمالي
- message: رسالة إضافية (اختياري)
- status: حالة الطلب
- whatsapp_link: رابط WhatsApp
- created_at: تاريخ الإنشاء
- updated_at: تاريخ التحديث
```

## ✏️ تعديل منتج موجود لإضافة خدمة WhatsApp:

1. اذهب إلى `/admin/products/{id}/edit`
2. ضع علامة على "يتطلب WhatsApp"
3. أدخل رقم WhatsApp
4. احفظ التغييرات

## 🚀 مثال على طلب API:

```javascript
fetch("/api/whatsapp-orders", {
    method: "POST",
    headers: {
        "Content-Type": "application/json",
        "X-CSRF-Token": "YOUR_TOKEN",
    },
    body: JSON.stringify({
        product_id: 1,
        customer_name: "محمد علي",
        customer_phone: "+966501234567",
        quantity: 2,
    }),
})
    .then((response) => response.json())
    .then((data) => {
        if (data.success) {
            window.open(data.whatsapp_link, "_blank");
        }
    });
```

## 📝 ملاحظات مهمة:

1. **رقم WhatsApp يجب أن يكون بصيغة دولية** (مثال: +966501234567)
2. جميع الطلبات يتم حفظها تلقائياً في قاعدة البيانات
3. يمكن تتبع حالة كل طلب من لوحة الإدارة
4. رسالة WhatsApp تشمل اسم العميل والكمية والسعر الإجمالي
5. العملاء سيتم توجيههم مباشرة إلى WhatsApp لإكمال الطلب
