# Setup Guide: نظام شحن الألعاب والتطبيقات

## خطوات التثبيت والإعداد

### الخطوة 1: البيانات الأساسية (مكتملة!)

✅ تم إنشاء الجدول والـ Model والـ Controller
✅ تم إضافة الفئة الجديدة "ألعاب وتطبيقات"
✅ تم إنشاء الواجهة الأمامية والإدارية

### الخطوة 2: إضافة منتجات الألعاب

1. اذهب إلى **لوحة الإدارة** → **المنتجات** → **إضافة منتج جديد**
2. أكمل البيانات:
    - الاسم: (مثلاً: "شحن رصيد Fortnite $50")
    - السعر: (السعر العادي)
    - الفئة: **ألعاب وتطبيقات**
    - الوصف: (وصف المنتج)
    - الصورة: (صورة اللعبة)
3. اضغط **حفظ**

### الخطوة 3: إعداد Telegram (اختياري ولكن موصى به!)

1. فتح Telegram وابحث عن **@BotFather**
2. أرسل `/newbot`
3. أتبع الخطوات واحصل على: **Bot Token**
4. لاحظ الـ Token (مثال: `123456789:ABCDEFGHijkl_MNOpqrstu`)
5. لحصول على **Chat ID**:
    - ابحث عن **@userinfobot**
    - أرسل أي رسالة
    - انسخ Chat ID (رقم طويل مثل: `987654321`)

6. افتح `.env` وأضف:

```
TELEGRAM_BOT_TOKEN=123456789:ABCDEFGHijkl_MNOpqrstu
TELEGRAM_CHAT_ID=987654321
```

### الخطوة 4: إعادة تشغيل الخادم

```bash
php artisan cache:clear
php artisan config:clear
```

## كيفية الاستخدام

### للعملاء (الزوار):

1. اذهب إلى **الألعاب والتطبيقات** من الصفحة الرئيسية
2. اختر اللعبة التي تريد شحنها
3. اضغط **🎮 شحن الرصيد**
4. أكمل النموذج:
    - أدخل معرف حسابك في اللعبة
    - ارفع صورة إثبات التحويل البنكي
    - أدخل رقم العملية
    - أكمل بيانات التواصل (اختياري)
5. اضغط **إرسال الطلب**

### للإدارة:

1. اذهب إلى **لوحة الإدارة** → **🎮 طلبات شحن الألعاب**
2. ستجد جميع الطلبات مع الحالة
3. اضغط على أي طلب لعرض التفاصيل كاملة
4. ستجد صورة الإثبات وكل البيانات
5. غيّر الحالة من "قيد الانتظار" إلى "قيد المعالجة" ثم "مكتمل"
6. عند التحديث ستتلقى إشعار على Telegram (إذا كنت قد أعددته)

## الحالات وشرحها

- 🟡 **قيد الانتظار** (Pending): طلب جديد لم تراجعه بعد
- 🔵 **قيد المعالجة** (Processing): تقوم بمعالجة الطلب
- 🟢 **مكتمل** (Completed): تم تنفيذ الطلب بنجاح
- 🔴 **مرفوض** (Rejected): تم رفض الطلب

## الإنذارات والتنبيهات

عند ورود طلب جديد:

### Telegram 📱

تتلقى رسالة فورية بـ:

- اسم اللعبة
- معرف اللاعب
- رقم العملية
- بيانات العميل
- رابط مباشر للطلب

### البريد الإلكتروني (اختياري)

يمكن إضافة:

1. تعديل `sendNotification()` في `GameRechargeController`
2. إضافة `dispatch(new SendGameRechargeNotification())`
3. إنشاء Mailable

### WhatsApp (اختياري)

يمكن استخدام:

- Twilio API
- WhatsApp Business API
- معالج مخصص

## استكشاف الأخطاء

### الطلب لم يتم حفظه:

- تحقق من صحة الصورة (أقل من 2MB)
- محسوم الصلاحيات للمجلد `storage/app/public/`

### لم تصل الإشعارات:

- تحقق من `TELEGRAM_BOT_TOKEN` و `TELEGRAM_CHAT_ID` في `.env`
- تشغيل `php artisan config:clear`
- تحقق من الـ Logs: `storage/logs/laravel.log`

### الصور لا تظهر:

- تأكد أن `storage:link` تم تشغيله: `php artisan storage:link`
- تحقق من صحة المسار في قاعدة البيانات

## ملاحظات أمان

✅ جميع الصور محمية وفي مجلد آمن
✅ التحقق من صيغة الملف (جزء من اللعمليات)
✅ حد أقصى لحجم الملف (2MB)
✅ فقط المسؤولين يمكنهم الوصول إلى لوحة التحكم
